# Git Tools

Tools and challenges for improving Git-based development workflows.

## Projects

### Clone & Just Works Challenge

A universal challenge: **For every source repo in the world, you should be able to clone it onto a clean machine, run the "build and test" script at the root, and it should just work.**

See [CLONE_AND_BUILD_CHALLENGE.md](CLONE_AND_BUILD_CHALLENGE.md) for details and a reusable prompt you can give to Claude Code to audit your own repositories.

## Contents

- **CLONE_AND_BUILD_CHALLENGE.md** — Challenge description and community prompt
- **build_and_test_matrix_categorized.md** — Analysis matrix with repo categorization
- **build_and_test_matrix_categorized.html** — Interactive HTML version
- **BUILD_TEST_AUDIT.md** — Detailed audit report with findings and recommendations
- **build_and_test_results.txt** — Raw test output from analysis
- **REPO_INVENTORY.txt** — Inventory of analyzed repositories

## Getting Started

1. Read [CLONE_AND_BUILD_CHALLENGE.md](CLONE_AND_BUILD_CHALLENGE.md) for context
2. Copy the prompt into Claude Code
3. Run it against your own repo collection
4. Share your results with #CloneAndJustWorks
